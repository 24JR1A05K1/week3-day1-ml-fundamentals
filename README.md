# week3-day1-ml-fundamentals

This project is about building a simple Machine Learning model to predict student marks using Linear Regression.

## Dataset
- Study Hours
- Attendance
- Assignments
- Marks

## Steps Done
- Data Exploration
- Train-Test Split
- Model Training (Linear Regression)
- Predictions
- Evaluation (MAE, R²)
- Visualization (Plotly)
