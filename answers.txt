PART 5 – LINEAR REGRESSION QUESTIONS
1.What is the slope?
The slope is represented by the coefficient values of the Linear Regression model. It shows how much the predicted marks change when a feature increases by one unit while keeping the other features constant.
2.What is the intercept?
The intercept is the starting value of the regression equation. It is the predicted marks when all feature values (Study Hours, Attendance, and Assignments) are zero.
3.What do they mean?
The slope indicates the influence of each feature on the target variable. A positive slope means marks increase as the feature increases. The intercept provides the baseline value from which predictions begin.
PART 8 – MODEL EVALUATION QUESTIONS
1.What is MAE?
MAE (Mean Absolute Error) is a metric used to measure prediction error. It calculates the average absolute difference between actual values and predicted values. A lower MAE indicates better model performance.
2.What is R˛ Score?
R˛ Score (Coefficient of Determination) measures how well the model explains the variation in the target variable. Its value ranges from 0 to 1. A value closer to 1 indicates a better-fitting model.
3.Is your model performing well?
Yes, if the MAE value is low and the R˛ Score is close to 1, the model is performing well. These values indicate that the predictions are close to the actual marks and the model explains most of the data variation.



1.What is Machine Learning?
Machine Learning is a branch of Artificial Intelligence (AI) that enables computers to learn from data and make decisions without being explicitly programmed for every task. Instead of following fixed instructions, machine learning algorithms identify patterns in data and use those patterns to make predictions or classifications. For example, recommendation systems used by Netflix and YouTube learn user preferences and suggest content accordingly. Machine learning involves collecting data, preparing it, selecting features, training models, evaluating performance, and making predictions. There are different types of machine learning such as supervised learning, unsupervised learning, and reinforcement learning. Supervised learning uses labeled data to train models, while unsupervised learning discovers hidden patterns in data. Machine learning is widely used because it can process large amounts of information quickly and accurately. As data continues to grow in today's digital world, machine learning has become one of the most important technologies for solving real-world problems and improving decision-making across industries.
2.What is Linear Regression?
Linear Regression is one of the simplest and most commonly used machine learning algorithms. It is used to predict a numerical value based on the relationship between one or more independent variables and a dependent variable. The algorithm works by finding the best-fitting straight line through the data points. This line is called the regression line. In a student performance dataset, for example, study hours, attendance, and assignments can be used to predict marks. Linear Regression calculates coefficients (slopes) and an intercept to create a mathematical equation for prediction. The coefficients show how much each feature affects the target variable, while the intercept represents the starting value. Linear Regression is easy to understand, simple to implement, and computationally efficient. It is commonly used in areas such as sales forecasting, salary prediction, stock analysis, and academic performance prediction. Because of its simplicity and interpretability, it is often the first algorithm learned by machine learning beginners.
3.What are Features?
Features are the input variables or independent variables used by a machine learning model to make predictions. They contain the information that helps the model understand patterns in the data. For example, in a student performance dataset, Study Hours, Attendance, and Assignments are features because they influence the student's marks. Features can be numerical values, text, categories, dates, or other types of data. Choosing the right features is an important step in machine learning because irrelevant features can reduce model performance. The process of selecting useful features is called feature selection. Good features help the model learn meaningful relationships and improve prediction accuracy. In real-world applications, features may include age, income, temperature, experience, or customer behavior depending on the problem being solved. Machine learning models analyze these features to identify patterns and generate predictions. Therefore, features serve as the foundation of any machine learning system and play a major role in determining the quality of results.
4.What are Labels?
Labels are the target values or outputs that a machine learning model is trained to predict. In supervised learning, labels are already known and provided in the training dataset. For example, in a student performance dataset, Marks is the label because it is the value being predicted using features such as Study Hours, Attendance, and Assignments. Labels help the model learn the relationship between inputs and outputs. During training, the model compares its predictions with the actual labels and adjusts itself to reduce errors. Labels can be numerical values, such as marks or salaries, or categories, such as spam or not spam. Accurate labels are important because incorrect labels can mislead the model and reduce prediction quality. In many machine learning projects, collecting and labeling data is one of the most time-consuming tasks. Labels provide the correct answers that guide the learning process and help the model make accurate predictions on new, unseen data.
5.What is the Difference Between Training Data and Testing Data?
Training data and testing data are two important parts of a machine learning dataset. Training data is used to teach the model and help it learn patterns and relationships between features and labels. During this phase, the model adjusts its parameters to improve prediction accuracy. Testing data, on the other hand, is used to evaluate the model after training. It contains unseen data that was not used during learning. This helps determine how well the model performs on new information. A common practice is to use 80% of the dataset for training and 20% for testing. If a model performs well on training data but poorly on testing data, it may be overfitting. Testing data provides an unbiased measure of model performance and ensures that the model can generalize to real-world situations. Both training and testing datasets are essential for building reliable and accurate machine learning systems.
6.Where is Machine Learning Used in Real Life?
Machine Learning is used in many real-life applications across different industries. In healthcare, it helps doctors diagnose diseases, analyze medical images, and predict patient outcomes. In finance, machine learning is used for fraud detection, credit scoring, and stock market analysis. E-commerce companies such as Amazon use recommendation systems to suggest products based on customer preferences. Social media platforms use machine learning to personalize content and advertisements. In transportation, machine learning powers self-driving cars and traffic prediction systems. Educational platforms use it to recommend learning resources and track student performance. Voice assistants like Siri, Alexa, and Google Assistant rely on machine learning to understand and respond to user commands. Machine learning is also widely used in cybersecurity, weather forecasting, agriculture, manufacturing, and entertainment. Its ability to learn from data and improve over time makes it a valuable technology for solving complex problems and improving efficiency in everyday life.
